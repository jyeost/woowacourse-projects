### 학습 목표

## 🌳 Level 4 : 팀 프로젝트 유지 보수 및 웹 백엔드에 깊이를 더하는 단계

### 기간

- 2023.08.29 ~ 2023.10.27

### 학습 목표

- 실사용자의 피드백을 받아 서비스의 완성도를 높이고 유지보수를 하는 경험을 한다.
